# ns-scripts

[![Node CI](https://github.com/natterstefan/ns-scripts/actions/workflows/ci.yml/badge.svg)](https://github.com/natterstefan/ns-scripts/actions/workflows/ci.yml)

## Setup

```bash
yarn
```

## Build

```bash
yarn build
```

## Start

```bash
# development
yarn dev

# production
node ./bin
```
